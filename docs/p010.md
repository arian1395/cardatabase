# Creating a project

## Steps
1. Open [Spring Initializr](https://start.spring.io)
2. Fill the left form
   *  Project (Maven | Gradle)
   *  Language (Java | Kotlin | Groovy)
   *  Spring Boot (2.6.9 | 2.7.1 | ... )
   * Project Metadata
      *  Group
      *  Artifact
      *  Name
      *  Description
      *  Package name
      *  Packaging (Jar | War)
      *  Java (18 | 17 | 11 | 8)
